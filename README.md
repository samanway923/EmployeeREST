# EmployeeREST
Work in progress to learn basic REST API development using an Employee management service demo.

BASE PATH: /employees

Features:
1. Adding a new employee (/createNew)
2. Deleting an employee by ID (/delete)
3. Searching for an employee by ID (/searchByid)
4. Searching for employees by location (/searchByLoc)
5. Listing all employees (/listAll)
